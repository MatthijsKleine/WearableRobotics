Linear Actuator

An actuator-driven system enabling fluid and lifelike motion.

Prototype Overview

This linear actuator, connected to a syringe, demonstrates the potential of low-pressure hydraulics in creating smooth, natural motion for wearable devices.

Key Features

  • Low-pressure hydraulics for safe and lifelike motion.
  • Easy integration into existing wearable systems.
  • Compact design for versatility in performances.
