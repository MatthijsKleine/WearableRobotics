Rack-and-Pinion System

A motion system for generating sinusoidal movement.

Prototype Overview

The 3D-printed rack-and-pinion mechanism demonstrates how sinusoidal motion can be integrated into wearable devices to create rhythmic effects for artistic performances.

Key Features

  • 3D-printed design for lightweight and durable performance.
  • Sinusoidal motion for rhythmic and hypnotic movements.
  • Simple integration into wearable robotics systems.

Created by Matthijs Kleine for the Wearable Robotics Toolkit Project.
